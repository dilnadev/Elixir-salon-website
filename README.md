# Elixir Salon Website

A luxury beauty salon website, currently built as a React + Vite application.

## Project Structure

- **`elixir-salon/`** — the live, actively maintained site (React + Vite + Tailwind CSS). This is the version that gets built and deployed.
- **`code.html`** — the original static HTML prototype from before the React conversion. Kept for historical reference only; it is no longer updated and is not required to run the site.
- **`DESIGN.md`** / **`screen.png`** — early design notes and a reference screenshot from the initial design phase.

## Getting Started

```bash
cd elixir-salon
npm install
npm run dev
```
